## Guide to text files
These symbols can be changed from `setting.py` file
```
# : Blocked Node
. : Walkable Node
S : Starting Node
E : Ending Node
@ : Path Node
```
-----------------------------

## How to use

Generator

```bash
python3 maze_gen.py
```

Path Finder

```bash
python3 solve.py maze.txt
```
